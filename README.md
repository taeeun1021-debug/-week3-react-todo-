# Week 3 React Todo 앱

## 📌 프로젝트 소개
이 프로젝트는 React와 JSX를 사용하여 만든 Todo 앱입니다.  
사용자는 할 일을 추가하고, 완료 여부를 체크하며, 삭제할 수 있습니다.  
또한 전체, 진행중, 완료 상태를 필터링하여 확인할 수 있습니다.

## 🛠️ 기능
- 할 일 추가
- 완료/진행중 체크
- 삭제
- 필터: 전체 / 진행중 / 완료
- 상태별 개수 표시 (전체, 완료, 진행중)

## ⚡ 사용 기술
- React 18 (CDN 방식)
- JSX
- Tailwind CSS (스타일링)
- Babel (JSX 변환)
